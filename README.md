# Task Manager
This repo works with `Python v3.10` and up.

## Setup & Installation

1. Clone the repo:
```bash
$ git clone <repo-url>
```
2. Create virtual environment:
```bash
$ virtualenv env
```
3. Run this command:
```bash
$ source ./env/bin/activate
```
4. Install the dependencies
```bash
$ (env) pip install -r requirements.txt
```
## Running The App
```bash
$ (env) python main.py
```
## Viewing The App

Go to `http://127.0.0.1:5000`
